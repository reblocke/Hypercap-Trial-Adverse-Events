# Contributing to Hypercap Trial Adverse Events

Thanks for your interest in contributing!

## How to contribute

- **Report bugs or request features**: Open an Issue on GitHub.  
- **Make changes**: Fork the repo, create a branch, and submit a Pull Request.  
- **Documentation**: Improve README, add comments, or expand examples.  
- **Analysis contributions**: Add new scripts, notebooks, or methods that align with the project scope.  

## Development guidelines

- Keep code modular and reproducible.  
- Add comments for clarity.  
- If adding dependencies, update `environment.yml`.  
- For analysis scripts, aim to map outputs clearly to manuscript figures/tables.  

## Community guidelines

- Be respectful and collaborative.  
- Document your changes clearly.  
- Follow the [Contributor Covenant](https://www.contributor-covenant.org/) for conduct.
